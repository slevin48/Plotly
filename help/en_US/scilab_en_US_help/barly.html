barly

Short description

Syntax

barly(y)

Arguments

y

scalar, vector of size N, or matrix N*M of real numbers or encoded integers, with

  • N : number of groups (each group of bars gathered on/around an x position)
  • M : number of categories in each group

Description

A simple bar chart

Examples

y=[1 -3 5];
barly(y);
