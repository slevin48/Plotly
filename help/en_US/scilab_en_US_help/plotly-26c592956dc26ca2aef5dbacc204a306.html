plotly

Short description

Syntax

plotly(x, y)

Arguments

x

vector or matrix of real numbers or integers.

y

vector or matrix of real numbers or integers.

Description

plotly plots a 2D curve and export it as an HTML file.

Examples

x=[0:0.1:2*%pi];
y=2*sin(x);
plotly(x,y)
